# TadobaLand

TadobaLand is a niche land marketplace focused on land listings around the Tadoba region.  
It allows land owners to list land for **sale, lease, or resort development**, and buyers to discover opportunities near Tadoba Tiger Reserve.

## Tech Stack

Frontend:
- React
- Vite
- TailwindCSS
- React Router

## Features (Current)

- Property listing grid
- Search by village / area
- Filters (type, area)
- Owner listings
- Agent listings
- Property carousel
